# lmlib Documentation

**Version:** {{VERSION}} — Open-Source and for Python.

Official documentation for [lmlib](https://pypi.org/project/lmLib/), the signal processing library using linear state space models for computationally efficient event detection, signal convolution or correlation, and signal decomposition tasks in time series (single- and multi-channel) and N-dimensional data (including images, videos, ...).

This library supports scipy, JIT (Just-in-Time) compilation and GPU accelerators.

Detailed information and scientific background are available in our publications (see [Bibliography](bibliography.md)).

**Typical Fields of Applications**:

- [Recursive least squares filters (options: weighted, windowed) ](_generated/examples/index.md#recursive-least-squares-filtering)
- [Recursive linear filters (convolutions and correlations)](_generated/examples/index.md#convolution)
- [Shape Detection](_generated/examples/index.md#shape-detection), [event and feature detection, likelihood testing](_generated/examples/index.md#event-detection-with-two-sided-line-models)
- for one-dimensional and [multi-dimensional data sets](_generated/examples/index.md#n-dimensional-signal-processing)
